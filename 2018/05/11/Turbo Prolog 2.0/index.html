Turbo Prolog 2.0

语法规范

  • 常量(symbol)小写，变量大写，这两点是强制性的，常量若大写需要加引号
  • 谓词小写，非强制，大写不会报错
  • PREDICATES, GOAL, CLAUSES顺序没有要求

数据类型

  • 串 string
    • 用一对双引号括住的任意字符序列。程序中的串最长可达 255 个字符，文件中的可为 64k
    • ‘abc’, ‘123’
  • 符号 symbol
    • 以小写字母打头的字母、数字和下划线序列；串
    • ‘abc’, abc
